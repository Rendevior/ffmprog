# FFmprog
FFmpeg Simple Progresss bar written in shell

## Installation
```
curl -sL -o "${PREFIX}/bin/ffmprog" "https://raw.githubusercontent.com/TheGlockMisc/ffmprog/master/ffmprog"
chmod +x "${PREFIX}/bin/ffmprog"
```

## Usage
```
./ffmprog -i [<url/file>] [args...]
```
## Dependencies
- bc
- ffmpeg
